Referensi Saya

html

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan merancang halaman web. HTML menggunakan tag atau elemen untuk mengatur struktur dan konten pada halaman web, seperti teks, gambar, tautan, dan media lainnya. Elemen-elemen ini membentuk kerangka dasar yang ditampilkan oleh browser, dan HTML bekerja bersama dengan CSS dan JavaScript untuk memberikan tampilan dan interaktivitas pada situs web

css

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak elemen-elemen pada halaman web yang dibuat dengan HTML. CSS memungkinkan pengembang untuk mengontrol desain, seperti warna, font, margin, padding, dan posisi, sehingga halaman web menjadi lebih menarik dan responsif. Dengan CSS, tampilan sebuah website bisa diatur secara konsisten dan efisien

css layout

CSS Layout adalah teknik yang digunakan untuk mengatur dan mengelola tata letak elemen-elemen pada halaman web. Dengan CSS layout, pengembang bisa mengontrol bagaimana elemen-elemen seperti teks, gambar, dan blok konten ditempatkan serta diatur dalam kolom, baris, atau grid pada halaman. Beberapa metode yang umum digunakan untuk layout adalah Flexbox, Grid, dan Float, yang membantu membuat desain web lebih responsif dan terorganisir
